功能亮点

简洁语法

Jvaw 采用了简洁的语法设计，减少了冗余代码，让开发者能够更专注于逻辑实现。

高性能

通过优化的编译器和运行时环境，Jvaw 能够提供出色的性能表现，满足各种高性能计算需求。

跨平台支持

Jvaw 支持多种操作系统和硬件平台，让开发者能够轻松地在不同环境中部署和运行代码。

快速开始

安装 Jvaw

在命令行中运行以下命令来安装 Jvaw：

git clone https://gitee.com/lgcr837/jvaw.git
cd jvaw

编写第一个程序

创建一个名为 hello.jvaw 的文件，并输入以下代码：

printn("Hello World!\nHello Jvaw!")

运行程序：

jvaw hello.jvaw

但因为项目还没有完善，所以你不会看到
Hello World!
Hello, Jvaw!
